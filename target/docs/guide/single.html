1 Introduction

MetriDoc is an extendable platform to view and maintain library statistics and reports. The platform is based on the web framework Grails which provides a simple web framework as well as a scripting engine that is well suited for various jobs that need to to be run in MetriDoc to transport data from internal systems to a central repository. Problem spaces are split into Grails plugins to enhance the user experience as well as encourage community plugins.

2 Getting Started

2.1 Required Software

  • Java 6 or above
  • Grails 2.0.4
  • Mysql
  • Subversion

Package managers generally have grails. For instance, if you are using a mac, try MacPorts and do sudo port install grails. For Ubuntu you can follow the instructions on the bottom of the Grails download page

2.2 First Metridoc Project

Although it is possible to create a project from scratch, there is a basic project already available. To use the base project you must have subversion installed. Move to a directory where you would like to install the metridoc workspace. Run the following commands to install and run the base Metridoc project

svn export http://metridoc.googlecode.com/svn/trunk/startup/metridoc
    cd metridoc
    grails run-app
